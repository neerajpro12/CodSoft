This calculator application was made in Python.

Calculator.py is made using Python GUI library "Tkinter".
Tkinter is a Python library that can be used to construct basic graphical user interface (GUI) applications.


It is a basic Calculator application which is capable of doing following operations:
 1. addition
 2. subtraction
 3. multiplication
 4. division

This application js also capable of displaying 'ERROR' on the display if any wrong unput is provided.

The application is capable of including floating numbers in the calculations.
